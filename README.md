# TS MAPS

Simple TypeScript Application. <br>
Defines a user and a company.<br>
Uses their location data to display it into google maps.<br>
User and company can be generated randomly.

#### Project structure

```
├── app
│   ├── src
│   │   ├── index.ts
│   │   ├── Map.ts
│   │   ├── User.ts
│   │   ├── Company.ts
│   ├── index.html

```
